# Customer Churn Analysis using Python

## Project Overview
This project analyzes customer churn data using Python and Jupyter Notebook. The objective is to identify the factors that influence customer churn and build insights from the dataset.

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
  

## Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Feature Engineering
- Model Building
- Model Evaluation

## Files
- Customer_Churn_Analysis.ipynb

## Results
The notebook includes visualizations and machine learning model evaluation for customer churn prediction.
